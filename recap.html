Aide mémoire des sélecteurs CSS déjà vus

  • * sélectionne TOUS les éléments de la page html
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les éléments ayant le nom de mon id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne les liens <a> inclus/imbriqués dans la balise <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background: couleur de fond
  • Permet de méttre une couleur de fond normal, en rgb, ici un exemple en linear-gradient
  1. color: couleur de la police
  • Permet de changer la couleur du texte
  1. float: effet flottant
  • permet de positionner un élément a gauche ou a droite de son élément parent
  1. font-family: police
  • Permet de choisir la typo ici un exemple avec la typo ('Courier New', Courier, monospace)
  1. font-size: taille de la police
  • Permet de choisir la taille d'un texte
  • height: hauteur de l'élément
  • Permet de choisir la hauteur d'un éléments
  • left: position à partir de la gauche de la fenêtre
  • Premet le décalage a partir gauche de la fenétre exemple left: 10px (l'élément sera a 10px du bord de la fenêtre
  • line-height: hauteur de l'interligne
  • Permet de méttre un espace entre les lignes
  • margin: marges extérieures
  • Permet de faire une marge entre l'élément enfant et l'élément parent exemple (margin: 10px;) l'enfant sera a 10px de marge partout de son parent
  • padding: marges intérieures
  • Le padding permet de définir les différents écarts de remplissage d'un élément, sur l'ensemble de ses côtés (cf. les boîtes CSS). Elle synthétise padding-top, padding-right, padding-bottom, padding-left.
  • position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
  • La position définit la façon dont un élément est positionné dans un document. Les propriétés top, right, bottom et left déterminent l'emplacement final de l'élément positionné.
  • text-align: alignement du texte
  • text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  • top: position à partir du haut de la fenêtre
  • transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  • width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em, ...)
